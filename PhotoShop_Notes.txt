{

1、使用矩形工具，拉出一个瘦长的矩形，填充颜色(Alt + delete)，取消选区(Ctrl + D)；
2、编辑--自由变换(Ctrl + T)：	（按住 Ctrl + alt + shift 三个键，拖动角点，可使图形变成梯形，按回车确定）

制作多个梯形：
3、复制图层：			 	     拖动图层到“新建图层的图标”上，可直接复制图层
4、自由变换(Ctrl + T)：			     可以进行缩放、倾斜、旋转等动作；拖动变换的轴心，可以控制形状的旋转方式。
5、一边复制一边再次变换(alt + ctrl + shift + T)：  编辑--变换--再次(ctrl + shift +T),重复上一次变换的操作，同时按下 Alt 键，可 一边复制图层一边再次变换
6、将多个图层合并：			     移动工具按住键盘的 ctrl 键，单击、拖拉图层，框住图层就都选中，图层--合并图层(Ctrl + E)

制作圆心：
7、制作外大圆：	  勾出标尺、参考线，右击选取工具，在圆心单击椭圆，并按住alt键可以得到同心变换的效果，再按住shift键可以得到一个同心变换的正圆。（切换矩形选框、圆形选框工具快捷键：shift + M） 		  新建图层，填充颜色，取消选区，合并图层（只要画心的像素都要新建图层）

9、制作内小圆：   圆形选框--从中心按住 alt+shift 键拖出一个椭圆，新建图层，填充颜色，取消选区，合并图层

制作弯弧梯形：
10、制作齿轮竖轴：选区工具--从中心按住 alt 键拖出一个矩形(不需要按 shift 键，不然会变成正方形)，新建图层，填充颜色，取消选区
11、制作齿轮横轴：复制矩形图层，编辑--自由变换--旋转90度--合并图层

12、吸管拾取前景色--按住 Ctrl 键 + 单击图层图标（得到当前图层所有像素范围的选区）-- 填充前景色，取消选区

抹去图形黑边：
13、单击图层面板上的锁定透明区域(第一个图层的上方第一个小按钮)，使透明区域不能被填充颜色，直接填充前景色即可，黑边消除
	注：Ctrl 键 + 单击图层图标 等到的图层选区会有黑边，建议使用“锁透明”方法对图层进行操作（把除想操作的图层全部变为不可见，锁上透明）

1、路径选择工具(小黑)：	   可选择编辑出的路径

钢笔工具（P）：
钢笔工具右下角图标：
1、为“X”代表绘制一条新的路径
2、什么也没有 代表正在画当前的这条路径
 
钢笔的运用：1、画了一条路径，想再画一条新的路径时，按 Ctrl 键在空白位置单机，能从上一路径中退出来，故又可画一条新的路径
	    2、画出曲线的路径，画完点不要松，拖拉鼠标，可控制路径的角度（除第一个锚点以外）	
	    3、画出转折的曲线路径，按住 Alt 键，画新点
	    4、选用钢笔工具画路径时要在属性栏选中“路径”的按钮图标(第二个)
	    5、钢笔工具下有一个“转换点工具”，能对已画出的路径上的点进行性质上的转换(平滑曲线变成转角曲线等等)
   	注：转换点工具与小白工具的区别：转换点工具是对本质上的编辑，而小白工具是根据已有的本质进行编辑

2、在小黑工具下，属性栏里，选择形状的“加、减、交叉等”
3、如 1、2这样依次把需要进行布尔运算的形状都设置好，然后点击属性栏的组合

图层：
{
图层样式窗口(双击图层):
(1)投影:	将为图层上的对象、文本或形状后面添加阴影效果。投影参数由"混合模式"、"不透明度"、"角度"、"距离"、"扩展"和"大小"等各种选项组成，通过对这些选项的设置可以得到需要的效果。

(2)内阴影:	将在对象、文本或形状的内边缘添加阴影，让图层产生一种凹陷外观，内阴影效果对文本对象效果更佳。

(3)外发光:	将从图层对象、文本或形状的边缘向外添加发光效果。设施参数可以让对象、文本或形状更精美。

(4)内发光:	将从图层对象、文本或形状的边缘向内添加发光效果。

(5)斜面和浮雕:	"样式"下拉菜单将为图层添加高亮显示和阴影的各种组合效果
①外斜面:    沿对象、文本或形状的外边缘创建三维斜面。

②内斜面:    沿对象、文本或形状的内边缘创建三维斜面。

③浮雕效果:  创建外斜面和内斜面的组合效果。

④枕状浮雕:  创建内斜面的反相效果，其中对象、文本或形状看起来下沉。

⑤描边浮雕:  只适用于描边对象，即在应用描边浮雕效果时才打开描边效果。

(6)光泽:	将对图层对象内部应用阴影，与对象的形状互相作用，通常创建规则波浪形状，产生光滑的磨光及金属效果。

(7)颜色叠加:	将在图层对象上叠加一种颜色，即用一层纯色填充到应用样式的对象上。从"设置叠加颜色"选项可以通过"选取叠加颜色"对话框选择任意颜色。

(8)渐变叠加:	将在图层对象上叠加一种渐变颜色，即用一层渐变颜色填充到应用样式的对象上。通过"渐变编辑器"还可以选择使用其他的渐变颜色。

(9)图案叠加:	将在图层对象上叠加图案，即用一致的重复图案填充对象。从"图案拾色器"还可以选择其他的图案。

(10)描边:	使用颜色、渐变颜色或图案描绘当前图层上的对象、文本或形状的轮廓，对于边缘清晰的形状(如文本)，这种效果尤其有用。

折叠编辑本段参数介绍:
(1)混合模式:	不同混合模式选项。

(2)色彩样本:	有助于修改阴影、发光和斜面等的颜色。

(3)不透明度:	减小其值将产生透明效果(0=透明，100=不透明)。

(4)角度:	控制光源的方向。

(5)使用全局光:	可以修改对象的阴影、发光和斜面角度(保证特效光源一致)

(6)距离:	确定对象和效果之间的距离。

(7)扩展/内缩:	"扩展"主要用于"投影"和"外发光"样式，从对象的边缘向外扩展效果;"内缩"常用于"内阴影"和"内发光"样式，从对象的边缘向内收缩效果。

(8)大小:	确定效果影响的程度，以及从对象的边缘收缩的程度。

(9)消除锯齿:	打开此复选框时，将柔化图层对象的边缘。

(10)深度:	此选项是应用浮雕或斜面的边缘深浅度

(11)混合选项默认：
混合颜色带：  	        使整个图层像素产生过滤或者溶解的效果    
本图层与下一图层：	滑动左边的色标，能控制图层像素溶解的效果，按住 Alt 键能使色标分解，能更好的控制溶解效果
		 注：不只是黑色能被溶解，只要是暗的颜色都能


文字图层转换为普通图层(可以填充像素)：  在文字的那一图层上，点击 右键--栅格化文字，就能是文字的图层与普通图层一样
选择图层：  	  【Alt】 + 【[】 键(光标下移)  【Alt】 + 【]】键(光标上移) 

图层的混合模式：  层与层的像素做计算，颜色是按照百分比来实现的，黑为 0，白为 255 ,百分比就是 0到1
滤色：   	  上一层对下一层像素的颜色做计算，使更亮  计算公式： 1-（1 - 图层一）*（1 - 图层二）   
	  举例：  拿百分比为0.5的颜色(值为128)举例：1-（1 - 0.5）*（1 - 0.5）== 0.75 == (255 - 128)/2 == 192    意思就是说：两个值为 128 的颜色做滤色，会得到 192 的颜色值

正片叠底：	  与滤色相反，使更暗      计算公式： 图层一 * 图层二
	  举例：  拿百分比为0.5的颜色(值为128)举例：0.5 * 0.5 == 0.25 == 64    意思就是说：两个值为 128 的颜色做正片叠底，会得到 64 的颜色值

混合模式各种样式总结(从上至下)：
1、特殊类
2、变暗
3、变亮
4、融合(对比加强)
5、反色
6、色调
}



通道
{
通道：
1：彩色通道
2：Alpha通道
3：专色通道
计算机存储像素颜色的原理：通过保存三原色(RGB)值来保存，每个原色能保存256个值(0到255)，决定强弱，通过三种原色的RGB值来表示最终的颜色，RGB三原色一共能表示
			  2^8 * 2^8 * 2^8 - 1 == 2^24 - 1 == 256 * 256 * 256 - 1 == 3个字节 == 16777215 种颜色，还有一个字节(0到255)用来表示透明的强弱

三原色混合:
 黑 - 黑 - 黑(0 - 0 - 0)(000000)： 	 黑

 白 - 黑 - 黑(255 - 0 - 0)(ff0000)： 	 红

 白 - 白 - 黑(255 - 255 - 0)(ffff00)： 	 黄

 黑 - 黑 - 白(0 - 0 - 255)(0000ff)： 	 蓝

 黑 - 白 - 黑(0 - 255 - 0)(00ff00)：	 绿

 黑 - 白 - 白(0 - 255 - 255)(00ffff)：	 青

 白 - 黑 - 白(255 - 0 - 255)(ff00ff)： 	 紫

 白 - 白 - 白(255 - 255 - 255)(ffffff)： 白


选择通道作为选区： Alt + 通道的快捷键(一般为 Ctrl +3 、Ctrl + 4 等等)

 魔棒工具(W)：	      		  点击某处，自动选择那一处颜色相近的像素形成选区，容差值(可容忍的像素颜色值的差)默认为 32，容差值大，选区则大，容差值小，选区则小	
 魔棒属性栏--连续：   		  勾上--对紧挨着的选区的像素的颜色做判断    不勾--对该图层没有挨着选区的像素的颜色做判断
 魔棒属性栏--调整边缘：
             平滑：  		  对选中的选区里的像素边缘变平滑效果（适用于边缘粗糙的选区像素）

 做阴影：			  
 1、先根据物体的阴影形状来框出一个选区，填充黑色，点击“滤镜”，选择“模糊”，选择“高斯模糊”，弹出高斯模糊窗口，根据预览图调节最下方的滚动条，选择好后点击确定（注：高斯模糊是根据图层来调      节的，你选中的是哪个图层，它就调节那个图层）
 
 2、选出需要制作阴影的对象，新建图层，把选出的选区拷贝到新建的图层上，再把拷贝后的图层复制一份，锁定透明，填充黑色  再次选中已填充好了的对象(Ctrl + 单机图层图	     标)，滤镜--高斯    模糊    ，根据需要的效果调节
					

 给物体换颜色：			  选择需要换颜色的那一图层，按 Ctrl + U 键，弹出“色相/饱和度”窗口，在色相那一栏，滑动色标滑块，选择自己想要的颜色，点击确定

 前、背景色互换：		  快捷键 X
 创建剪切蒙版(Alt + Ctrl + G)：   上面图层里的像素放在下面图层像素里面去（上色下形），再次点击能还原(释放剪切蒙版)
